# 💎 MILAN JEWEL — Ювелирный Интернет-Магазин

Элегантный и функциональный Landing Page для бренда украшений премиум-класса. Проект сочетает в себе эстетичный дизайн и практичные инструменты для повышения продаж.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/MilanJewel.com/)

---

### 🌟 Ключевые особенности:
* **Маркетинговый таймер:** Встроенный счетчик обратного отсчета для стимулирования продаж в период акций.
* **Доступность (Accessibility):** Использование семантической верстки и ARIA-ролей для корректной работы скринридеров.
* **Стильная витрина:** Адаптивные карточки товаров с использованием Flexbox и Grid.
* **Типографика:** Интеграция шрифта *Barlow Semi Condensed* для создания современного минималистичного образа.

---

### 🛠 Стек технологий:
* **HTML5:** Семантическая разметка (`section`, `article`, `nav`).
* **CSS3:** Сложные сетки, адаптивность, стилизация форм поиска.
* **JavaScript (Vanilla):** Логика работы таймера и навигации.

---
